Formulários

Formulário com GET

Esse método envia os dados do formulário como parêmetros na URL. Especialmente usável em requisições de buscas, já que haverá um link onde requisição do formulário já foi respondida

Formulário com POST

Esse método envia os dados colocados no formulário via no corpo da requisição, útil para formulários de login, por exemplo.
