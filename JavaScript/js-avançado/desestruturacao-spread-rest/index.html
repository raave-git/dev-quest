Desestruturação, método spread e rest

spread

O operador de espalhamento é uma funcionalidade que permite expandir objetos iteráveis, como arrays ou strings, em lugares onde se esperam zero ou mais argumentos (para chamadas de funções) ou elementos (para arrays literais).

  1. Chamadas de Funções:

    Quando usamos o operador de espalhamento em chamadas de funções, ele permite que um objeto iterável (como um array) seja expandido para ser usado como argumentos da função. Exemplo:

                            
                            function minhaFuncao(x, y, z) {}
                            const args = [0, 1, 2];
                            minhaFuncao(...args);

    Antes, se usava a Function.prototype.apply para passar os elementos de um array como argumentos para uma função. Agora, o operador de espalhamento simplifica essa tarefa.

  2. Arrays Literais e Strings

    Podemos usar o operador de espalhamento para criar novos arrays ou strings combinando elementos de outros iteráveis. Exemplo:

                        
                        const arrayOriginal = [1, 2, 3];
                        const novoArray = [...arrayOriginal, 4, 5, 6];

    Aqui, novoArray conterá [1, 2, 3, 4, 5, 6].

  3. Espalhamento em Objetos Literais

    O operador de espalhamento também pode ser usado para criar objetos literais combinando propriedades de outros objetos. Exemplo:

                        
                        const objetoOriginal = { a: 1, b: 2 };
                        const objetoClonado = { ...objetoOriginal, c: 3 };

    Agora, objetoClonado terá as propriedades { a: 1, b: 2, c: 3 }

rest

O parâmetro rest permite representar um número indefinido de argumentos como um array. Ele é usado principalmente em duas situações:

  1. Chamadas de Funções:

    Quando usamos o parâmetro rest em chamadas de funções, ele nos permite aceitar um número variável de argumentos e tratá-los como um array dentro da função. Exemplo:

                        
                        function minhaFuncao(x, y, ...outrosArgs) {
                            // Faça algo com x, y e outrosArgs (que é um array)
                        }

                        minhaFuncao(1, 2, 3, 4, 5);

    Aqui, outrosArgs conterá [3, 4, 5].

  2. Arrays Literais e Strings:

    O parâmetro rest também pode ser usado para criar novos arrays ou strings combinando elementos de outros iteráveis. Exemplo:

                        
                        const arrayOriginal = [1, 2, 3];
                        const novoArray = [...arrayOriginal, 4, 5, 6];

    Agora, novoArray conterá [1, 2, 3, 4, 5, 6].

Em resumo, o parâmetro rest é uma maneira flexível de lidar com um número variável de argumentos em funções, permitindo que os argumentos sejam tratados como um array.
