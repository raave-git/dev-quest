Tags de divisão:

Sobre a tag div.

Essa tag geralmente é usada para conter um grupo de tags, isto é uma parte da página.

Sobre a tag span

Por outro lado, span é melhor para conter uma parte do texto, como essa.

Uma explicação mais técnica é que a tag div é uma tag de nivel bloco, por outro lado, a tag span é de nível linha. Isto é, block level e inline level.
