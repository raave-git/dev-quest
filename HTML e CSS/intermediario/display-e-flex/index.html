Display

Display block

Quando um elemento é definido como display: block, ele ocupa toda a largura disponível na página e cria uma nova linha abaixo dele. Isso significa que outros elementos não podem ser posicionados ao lado dele.

A altura e a largura do elemento podem ser definidas usando as propriedades CSS height e width.

Display inline

Quando um elemento é definido como display: inline, ele ocupa apenas o espaço necessário para o conteúdo dentro dele e não cria uma nova linha abaixo dele. Isso significa que outros elementos podem ser posicionados ao lado dele.

A altura e a largura do elemento não podem ser definidas usando as propriedades CSS height e width.

Display inline-block

Quando um elemento é definido como display: inline-block, ele ocupa apenas o espaço necessário para o conteúdo dentro dele e não cria uma nova linha abaixo dele, assim como um elemento definido como display: inline.

Ao contrário de um elemento display: inline, um elemento display: inline-block pode ter uma altura e largura definidas usando as propriedades CSS height e width, assim como um elemento display: block.

Display none

Quando um elemento é definido como display: none, ele não é exibido na página e não ocupa nenhum espaço na página. Isso significa que o elemento não é visível para o usuário e não pode ser interagido com.

A altura e a largura do elemento não podem ser definidas usando as propriedades CSS height e width.
